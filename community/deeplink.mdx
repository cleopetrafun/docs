Our partners can seamlessly integrate Cleopetra deep link on their platform or community to redirect users for creating LP positions in a specific token or pool.

<Tip>Partners earn boosted fee share from users onboarded through referral deep link. Contact us on TG for a custom link!</Tip>

You can add either one of these custom fields in your reflink:

## Token CA

Mint address of a Solana token. Replace `CODE` with your referral code and `ADDRESS` with token address.

```
https://t.me/cleopetra_bot?start=ref_CODE-ca_ADDRESS
```

## Pool Address 

Address of a specific Meteora DLMM pool. Replace `CODE` with your referral code and `ADDRESS` with token address.

```
https://t.me/cleopetra_bot?start=ref_CODE-pl_ADDRESS
```
